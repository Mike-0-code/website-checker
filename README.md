# ¿Está Caído? - Website Checker

Verifica si un sitio web está online directamente desde tu navegador.

## Características

- Verificación instantánea desde tu navegador
- Historial de verificaciones recientes
- Completamente privado - sin servidores
- Diseño responsive

## Cómo usar

1. Ingresa una URL (ejemplo: https://google.com)
2. Haz clic en "Verificar"
3. Ve el resultado: Online u Offline

## Demo

https://mike-0-code.github.io/website-checker/

## Tecnologías

HTML, CSS y JavaScript vanilla.

## Instalación

Abre index.html en tu navegador.
